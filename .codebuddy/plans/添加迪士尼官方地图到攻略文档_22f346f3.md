## Product Overview

在现有的上海迪士尼旅游攻略文档中添加官方地图图片，丰富攻略的视觉信息和实用性。

## Core Features

- 获取并插入高清的上海迪士尼乐园官方地图
- 在"园区导览"或"实用信息"章节展示地图
- 添加地图使用说明和重要地标介绍
- 确保地图清晰度、比例及文字可读性符合文档要求

## Tech Stack

- 文档处理: [skill:docx] 用于读取和修改现有攻略文档
- 图片处理: 标准图片插入与格式调整

## Tech Architecture

### System Architecture

- 架构模式: 单次文档处理流程
- 流程: 读取文档 -> 定位章节 -> 插入地图 -> 添加说明文字 -> 保存文档

### Data Flow

读取 .docx 攻略文档 -> 定位目标章节位置 -> 插入地图图片及说明 -> 保存更新后的文档

## Implementation Details

### Key Code Structures

**文档处理逻辑**: 使用 docx skill 打开现有文档，定位到指定文本节点（如"园区导览"），在之后插入图片对象，并附带段落说明。

**图片插入规范**: 确保图片宽度适应页面，分辨率不低于 300DPI，保持纵横比。

## Agent Extensions

### Skill

- **docx**
- Purpose: 读取现有的迪士尼旅游攻略文档，定位到"园区导览"或"实用信息"章节，插入官方地图图片并添加说明文字
- Expected outcome: 成功生成包含高清地图及使用说明的更新版攻略文档
