# 导航条两行格式调整计划

## 问题分析

当前导航条的日期项内容都在同一行显示，用户希望调整为明确的两行格式：

* 第一行：Day x + 日期数字（例如："Day 1 19"）

* 第二行：农历日期 + 地点（例如："正月初三 抵达杭州"）

## 解决方案

1. **调整布局结构**：

   * 使用CSS将日期项分为两个明确的行

   * 第一行：Day标签和日期数字居中显示

   * 第二行：农历日期和地点居中显示

   * 确保两行内容清晰分离

2. **修改CSS样式**：

   * 为Day标签和日期数字添加容器或使用CSS规则确保它们在同一行

   * 为农历日期和地点添加容器或使用CSS规则确保它们在同一行

   * 调整行高和间距，确保两行清晰分离

   * 保持内容居中对齐

3. **优化不同状态下的显示**：

   * 普通状态

   * 当天日期状态

   * 选中状态

   * 确保所有状态下都保持两行格式

4. **保持响应式设计**：

   * 在不同屏幕尺寸下都能正确显示两行格式

   * 调整移动端的布局和字体大小

## 具体修改内容

```css
/* 日期项样式 */
.date-item {
    text-align: center;
    /* 其他样式保持不变 */
}

/* 确保Day标签和日期数字在第一行 */
.day-label {
    display: inline-block;
    /* 其他样式保持不变 */
}

.date-number {
    display: inline-block;
    /* 其他样式保持不变 */
}

/* 确保农历日期和地点在第二行 */
.lunar-date {
    display: inline-block;
    /* 其他样式保持不变 */
}

.date-location {
    display: inline-block;
    /* 其他样式保持不变 */
}

/* 调整行内元素的间距 */
.day-label + .date-number {
    margin-left: var(--td-spacer-2);
    margin-right: var(--td-spacer-2);
}

.lunar-date + .date-location {
    margin-left: var(--td-spacer-2);
}
```

## 优化效果

* 导航条日期项显示为两行格式：

  * 第一行：Day x + 日期数字

  * 第二行：农历日期 + 地点

* 内容完整，清晰可读

* 保持良好的视觉层次

* 所有状态下都能正确显示

* 响应式设计适配不同屏幕尺寸

通过以上修改，导航条将按照用户要求的格式显示，分为明确的两行，同时保持内容的完整性和可读性。
