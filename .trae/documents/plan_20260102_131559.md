# 导航栏日期项高度优化计划

## 问题分析
当前导航栏的日期项高度较高，导致一屏幕无法显示所有7天行程。用户希望降低每个方格子的高度，尽可能在一屏幕内显示全部行程。

## 解决方案
1. **减少日期项内边距**：
   - 将 `.date-item` 的内边距从 `var(--td-spacer-5)` (20px) 改为更小的值
   - 保持足够的留白，确保视觉效果

2. **减小字体大小**：
   - 减小 `.date-number` 的字体大小
   - 减小 `.lunar-date` 的字体大小
   - 减小 `.date-location` 的字体大小

3. **减小元素间距**：
   - 减小 `.date-number` 的 `margin-bottom`
   - 减小 `.lunar-date` 的 `margin-bottom`
   - 减小 `.date-grid` 的 `gap` 间距

4. **调整行高**：
   - 减小 `.date-location` 的 `line-height`
   - 确保文字紧凑但清晰可读

## 具体修改内容

```css
/* 日期网格样式 */
.date-grid {
    display: grid;
    gap: var(--td-spacer-3); /* 从16px改为12px */
}

/* 日期项样式 */
.date-item {
    padding: var(--td-spacer-3); /* 从20px改为12px */
    background: linear-gradient(135deg, #ffffff 0%, #f5f7fa 100%);
    border: 1px solid var(--td-gray-color-3);
    border-radius: var(--td-radius-large);
    cursor: pointer;
    transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
    text-align: center;
    box-shadow: var(--td-shadow-1);
    position: relative;
    overflow: hidden;
}

.date-number {
    font-size: 20px; /* 从24px改为20px */
    font-weight: 700;
    margin-bottom: var(--td-spacer-1); /* 从8px改为4px */
    line-height: 1;
}

.lunar-date {
    font-size: 11px; /* 从13px改为11px */
    margin-bottom: var(--td-spacer-2); /* 从12px改为8px */
    opacity: 0.9;
    font-weight: 500;
}

.date-location {
    font-size: 12px; /* 从14px改为12px */
    line-height: 1.3; /* 从1.5改为1.3 */
    font-weight: 600;
}
```

## 优化效果
- 降低每个日期项的高度
- 减小日期项之间的间距
- 保持文字清晰可读
- 确保所有7天行程能在一屏幕内显示
- 保持良好的视觉效果和交互体验

通过以上修改，导航栏将更加紧凑，能够在一屏幕内显示所有行程，同时保持良好的可读性和视觉效果。
