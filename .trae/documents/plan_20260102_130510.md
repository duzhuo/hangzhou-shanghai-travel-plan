# 修复导航条选中状态文字显示问题

## 问题分析
当前导航条选中状态的文字几乎不可见，原因是：
1. 选中状态使用了绿色渐变背景
2. 文字颜色设置为白色
3. 背景色和文字色对比度不足，导致文字看不清楚

## 解决方案
1. **调整选中状态的配色方案**：
   - 更换选中状态的背景色，使用更浅的绿色或者不同的配色
   - 确保文字和背景色有足够的对比度
   - 保持良好的视觉效果

2. **具体修改内容**：
   - 修改 `.date-item.selected` 的背景色和文字颜色
   - 确保选中状态的文字清晰可见
   - 保持与整体设计风格的一致性
   - 优化选中状态的视觉效果

3. **修改方案**：
   - 方案一：使用蓝色系配色，与当前日期样式区分
   - 方案二：使用更深的绿色背景，调整文字颜色为白色
   - 方案三：使用边框高亮+背景色组合，文字颜色保持深色

## 推荐方案
采用方案一：使用蓝色系配色，与当前日期的品牌色区分，同时保持良好的对比度

```css
/* 选中日期样式 */
.date-item.selected {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    border-color: #667eea;
    color: white;
    box-shadow: var(--td-shadow-4);
    transform: translateY(-2px) scale(1.02);
}
```

或者方案三：使用边框高亮+浅色背景，文字保持深色

```css
/* 选中日期样式 */
.date-item.selected {
    background: linear-gradient(135deg, #e0f7fa 0%, #ffffff 100%);
    border: 2px solid var(--td-success-color);
    color: var(--td-text-color-primary);
    box-shadow: var(--td-shadow-3);
    transform: translateY(-2px) scale(1.02);
}
```

我将选择方案三，这样可以确保文字清晰可见，同时保持良好的视觉效果。
