# 导航条长度调整计划

## 问题分析
当前导航条的高度是根据内容自动调整的，从截图来看，需要将导航条的长度向下延伸到红色选框标记的位置，确保导航条与红框边界精确对齐。

## 解决方案
1. **调整导航条高度**：
   - 将 `.quick-nav` 的 `height` 从 `fit-content` 改为固定高度或 `min-height`
   - 确保导航条能够延伸到红色选框位置
   - 保持 `position: sticky` 效果

2. **调整内部布局**：
   - 确保日期网格内容正常显示
   - 调整内部元素的布局，保持视觉连贯性
   - 确保导航条底部与红框边界精确对齐

3. **响应式设计调整**：
   - 确保在不同屏幕尺寸下导航条都能正确显示
   - 调整移动端的导航条高度
   - 保持功能不受影响

## 具体修改内容

```css
/* 左侧快速导航条 */
.quick-nav {
    width: 280px;
    flex-shrink: 0;
    background: linear-gradient(135deg, #ffffff 0%, #f8f9ff 100%);
    border-radius: var(--td-radius-large);
    box-shadow: var(--td-shadow-3);
    padding: var(--td-spacer-6);
    position: sticky;
    top: var(--td-spacer-6);
    height: calc(100vh - 48px); /* 调整为合适的高度，确保延伸到红框位置 */
    overflow-y: auto;
    overflow-x: hidden;
    transition: all 0.3s ease;
}
```

## 优化效果
- 导航条长度延伸到红色选框标记位置
- 保持视觉连贯性和完整性
- 确保在不同屏幕尺寸下正常显示
- 功能不受影响
- 保持良好的用户体验

通过以上修改，导航条将延伸到指定位置，同时保持良好的视觉效果和响应式表现。
