# 导航条添加Day x显示计划

## 问题分析
当前导航条日期项显示两行内容：日期数字、农历日期+地点。用户希望在导航条中添加"Day x"显示，需要在保持两行格式的同时，合理布局Day信息，确保内容完整清晰，不影响可读性。

## 解决方案
1. **调整HTML结构**：
   - 在每个日期项中添加"Day x"元素
   - 将Day信息与日期数字组合在第一行
   - 保持农历日期+地点在第二行

2. **修改CSS样式**：
   - 调整日期数字和Day信息的布局
   - 确保两行格式保持不变
   - 优化字体大小和间距
   - 保持良好的信息层次

3. **更新所有状态样式**：
   - 普通状态
   - 选中状态
   - 当天日期状态
   - 确保所有状态下都能正确显示Day信息

4. **响应式设计调整**：
   - 确保在不同屏幕尺寸下都能正确显示
   - 调整移动端的布局和字体大小
   - 保持功能不受影响

## 具体修改内容

1. **修改HTML结构**：
   ```html
   <div class="date-item" data-day="day-1" data-date="2026-02-19">
       <div class="day-label">Day 1</div>
       <div class="date-number">19</div>
       <div class="lunar-date">正月初三</div>
       <div class="date-location">抵达杭州</div>
   </div>
   ```

2. **修改CSS样式**：
   ```css
   /* 添加Day标签样式 */
   .day-label {
       font-size: 12px;
       font-weight: 600;
       color: var(--td-brand-color);
       margin-bottom: 4px;
       display: block;
   }
   
   /* 调整日期数字样式 */
   .date-number {
       font-size: 20px;
       font-weight: 700;
       margin-bottom: var(--td-spacer-2);
       line-height: 1;
       display: block;
   }
   ```

3. **更新选中状态样式**：
   ```css
   /* 确保选中状态下Day标签颜色正确 */
   .date-item.selected .day-label {
       color: #000000 !important;
   }
   
   /* 确保当天日期状态下Day标签颜色正确 */
   .date-item.today .day-label {
       color: white !important;
   }
   ```

## 优化效果
- 导航条显示"Day x"信息
- 保持两行格式
- 信息层次清晰，可读性良好
- 所有状态下都能正确显示
- 响应式设计适配不同屏幕尺寸

通过以上修改，导航条将显示"Day x"信息，同时保持良好的视觉效果和用户体验。
