# 导航条日期项两行文本调整计划

## 问题分析
当前导航条日期项显示三行文本：日期数字、农历日期、地点。用户希望将其合并为两行格式，确保内容完整性和可读性不受影响，保持语义连贯性和信息准确性。

## 解决方案
1. **调整布局结构**：
   - 将日期数字作为第一行（保持大字体，突出显示）
   - 将农历日期和地点合并为第二行（使用较小字体，紧凑排列）
   - 保持良好的信息层次和可读性

2. **修改CSS样式**：
   - 调整 `.date-number` 样式，确保独占一行
   - 调整 `.lunar-date` 和 `.date-location` 样式，使其在同一行显示
   - 调整字体大小和间距，优化视觉效果
   - 确保选中状态下的样式同步更新

3. **保持响应式设计**：
   - 确保在不同屏幕尺寸下都能正确显示两行格式
   - 调整移动端的布局和字体大小
   - 保持功能不受影响

## 具体修改内容

```css
/* 日期项样式 */
.date-item {
    padding: var(--td-spacer-3);
    background: linear-gradient(135deg, #ffffff 0%, #f5f7fa 100%);
    border: 1px solid var(--td-gray-color-3);
    border-radius: var(--td-radius-large);
    cursor: pointer;
    transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
    text-align: center;
    box-shadow: var(--td-shadow-1);
    position: relative;
    overflow: hidden;
}

/* 日期数字样式 - 独占一行 */
.date-number {
    font-size: 24px;
    font-weight: 700;
    margin-bottom: var(--td-spacer-2);
    line-height: 1;
    display: block;
}

/* 农历日期样式 - 与地点在同一行 */
.lunar-date {
    font-size: 12px;
    margin-right: var(--td-spacer-2);
    opacity: 0.9;
    font-weight: 500;
    display: inline;
}

/* 地点样式 - 与农历日期在同一行 */
.date-location {
    font-size: 12px;
    line-height: 1.3;
    font-weight: 600;
    display: inline;
}
```

## 优化效果
- 日期项从三行变为两行，更加紧凑
- 保持了良好的信息层次（日期数字突出显示）
- 内容完整性和可读性不受影响
- 语义连贯性和信息准确性得以保持
- 在不同屏幕尺寸下都能正确显示

通过以上修改，导航条日期项将显示为两行格式，日期数字独占一行，农历日期和地点合并为一行，实现了用户的要求。
