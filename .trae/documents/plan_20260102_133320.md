# 导航条两行格式调整计划

## 问题分析
当前导航条显示三行内容：Day x、日期数字、农历+地点。用户希望将其重新调整为两行格式，确保内容完整且清晰可读。

## 解决方案
1. **调整布局结构**：
   - 将Day标签和日期数字合并到同一行
   - 保持农历+地点在第二行
   - 使用Flex布局实现第一行的左右或上下排列

2. **修改CSS样式**：
   - 将Day标签和日期数字设置为同一行显示
   - 调整字体大小和间距
   - 确保内容紧凑但清晰可读
   - 保持良好的视觉层次

3. **优化不同状态下的显示**：
   - 普通状态
   - 当天日期状态
   - 选中状态
   - 确保所有状态下都保持两行格式

4. **保持响应式设计**：
   - 在不同屏幕尺寸下都能正确显示两行格式
   - 调整移动端的布局和字体大小

## 具体修改内容

```css
/* Day标签样式 */
.day-label {
    font-size: 12px;
    font-weight: 600;
    color: var(--td-brand-color);
    margin-right: var(--td-spacer-2);
    display: inline;
}

/* 日期数字样式 */
.date-number {
    font-size: 24px;
    font-weight: 700;
    margin-bottom: var(--td-spacer-2);
    line-height: 1;
    display: inline;
}

/* 确保Day标签和日期数字在同一行 */
.date-item {
    text-align: center;
}

/* 调整第一行的布局 */
.date-item .day-label + .date-number {
    margin-bottom: var(--td-spacer-2);
}
```

## 优化效果
- 导航条恢复两行格式：
  - 第一行：Day x + 日期数字
  - 第二行：农历日期 + 地点
- 内容完整，清晰可读
- 保持良好的视觉层次
- 所有状态下都能正确显示
- 响应式设计适配不同屏幕尺寸

通过以上修改，导航条将恢复为两行格式，同时保持内容的完整性和可读性。
